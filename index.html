Welcome to Swing

Swing is a Python/JavaScript framework for putting cognitive science and psychology experiments. It can select the trials to be run in an experiment from a .CSV file, shuffle these trials and present them as a single web page, using PsiTurk, the "open platform for science on Amazon Mechanical Turk", to deploy the experiment and collect data from online participants through Amazon Mechanical Turk. Swing can be used in many fields of science; however, it is specifically designed for studies in cognitive science and psychology.
